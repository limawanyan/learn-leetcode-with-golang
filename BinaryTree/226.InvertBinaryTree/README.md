# 题目
翻转一棵二叉树。
示例：

输入：

        4
      /    \
     2      7
    / \   / \
    1   3 6   9
输出：
 
       4
     /   \
     7     2
    / \   / \
    9   6 3   1

# 解题方法
### 递归 DFS 前中后序遍历
    DFS 深度优先遍历
    时间复杂度:O(n) n为二叉树节点数目
    空间复杂度:O(n) n为递归的深度 平均情况下是为O(logN) 最坏情况下树成链状，所以空间复杂度为O(n)

### 迭代 BFS
    BFS 广度优先遍历
    层层扫荡，用队列临时存储遍历元素，不断迭代队列元素交换左右子树
    时间空间复杂度O(n)
